philosophy.md

Make it useful. Keep it weird.

Code does not have to be perfect to be worth sharing. It has to be honest, readable, testable, and useful enough that someone else can pick it up, break it, fix it, and make it their own.

The vision can change. The philosophy should not: build things, learn from the ugly parts, and leave the repo better than it was yesterday.
